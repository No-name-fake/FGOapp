# FGO Party Manager

> [!NOTE]
> **本プロジェクトの目的について**
> このアプリケーションは、FGOのプレイ体験をより便利にするために作成された個人用管理ツールです。

Fate/Grand Order (FGO) のパーティ編成を詳細に記録し、自分だけの編成ライブラリを作成・管理できるウェブアプリケーションです。
外部の最新データ（Atlas Academy API）と連携しているため、新登場のサーヴァントや概念礼装もすぐに反映されます。

## ✨ 主な機能

- **サーヴァント・礼装の所持管理**: 自分の所持状況をチェックしておくことで、編成を組む際に持っているキャラだけで絞り込みが可能です。
- **詳細なパーティ保存**: キャラクターと礼装の組み合わせだけでなく、スキルレベル(10/10/10)や宝具レベル、魔術礼装まで含めて保存できます。
- **周回・攻略メモ**: 「どのクエスト用か」「各ターンでどのスキルを使うか」といった立ち回りをメモとして残せます。
- **編成の一覧・共有**: 保存した編成を一覧で確認でき、自分以外のユーザーが作成した編成の閲覧も可能です。
- **管理者用ツール**: ユーザーの管理や、データのメンテナンスを行うための専用機能も備えています。

## 🛠 準備とインストール

### 1. 必要なパッケージのインストール
Pythonが使える環境で、以下のコマンドを実行して準備をします。

```bash
pip install -r requirements.txt
```

### 2. 設定ファイルの準備
アプリを正しく動かすための設定を行います。

1. ` .env.example` ファイルをコピーし、ファイル名を `.env` に変更します。
2. `.env` ファイルをテキストエディタで開き、`SECRET_KEY`（セキュリティ用の合言葉）などを設定します（デフォルトのままでも動作は可能です）。

## 🚀 アプリの起動方法

1. **プログラムの実行**
   ```bash
   python app.py
   ```
2. **ブラウザでアクセス**
   起動すると「Running on http://127.0.0.1:5000」と表示されます。
   ブラウザで [http://127.0.0.1:5000](http://127.0.0.1:5000) を開いてください。

## 🔧 管理者向けの操作コマンド
特別な設定をしたい場合は、コマンドプロンプトやターミナルで以下のコマンドが使用できます。

- **新しく管理者ユーザーを作る（または既存ユーザーを管理者に変更）**
  ```bash
  flask admin create [ユーザー名] [パスワード]
  ```
- **管理者のリストを確認する**
  ```bash
  flask admin list
  ```

## 📬 追加機能: メール送信（お問い合わせ / パスワードリセット）

以下の機能は、Gmailの設定を行うことで有効になります。  
**設定なしでも動作します**（その場合、メール内容はサーバーのコンソールに出力されます）。

### 対応機能

| 機能 | 説明 |
|---|---|
| お問い合わせ | フォームから送信した内容が管理者のメールアドレスに届きます |
| パスワードリセット | 登録メールアドレスにリセット用リンク（1時間有効）が届きます |

### 設定手順

#### 1. Googleアプリパスワードの発行

メール送信には通常のパスワードではなく、**Googleアプリパスワード（16桁）**が必要です。

1. [Googleアカウントのセキュリティページ](https://myaccount.google.com/security) を開く
2. **「2段階認証プロセス」を有効にする**（未設定の場合）
3. **「アプリパスワード」を選択** → 任意の名前（例: `FGO App`）を入力
4. 表示された **16桁のパスワード** をコピーする

#### 2. `.env` ファイルへの設定追加

`.env` ファイルに以下の3行を追記してください。

```env
SMTP_USER=your_gmail@gmail.com        # 送信元のGmailアドレス
SMTP_PASSWORD=xxxx xxxx xxxx xxxx    # アプリパスワード（16桁）
ADMIN_EMAIL=admin@example.com         # お問い合わせメールを受け取るアドレス
```

#### 3. サーバーを再起動して完了

```bash
python app.py
```

これだけで、お問い合わせフォームとパスワードリセットメールが実際に送信されるようになります。

> **ヒント**: メールアドレスを登録していないユーザーはパスワードリセットが使えません。  
> 登録時（新規登録ページ）にメールアドレスを入力するか、管理者が直接DBを編集して追加することができます。

## 🗺 今後の開発予定

現在は基本的な編成管理機能のみ実装されていますが、以下の機能追加を検討しています。

### 1. 編成の共有・閲覧機能

現在は自分が登録した編成しか見ることができませんが、**他ユーザーが作成した編成を閲覧できる共有機能**を予定しています。

- 自分の編成を「公開 / 非公開」で設定できるようにする
- 全体の公開編成一覧を閲覧できるページの追加
- 編成へのコメントや参考表示など

### 2. サーヴァントの詳細情報登録

現在の所持サーヴァント管理は「持っているか / いないか」の2択のみですが、**各サーヴァントの育成状況をより詳細に記録**できるように拡張を予定しています。

- スキルレベル(1〜10)、
- アペンドスキルレベル(1〜10)、
- 宝具レベル (1〜5)

これらを事前に登録しておくことで、**編成を組む際に自動的に反映され、毎回手動で入力し直す手間を省く**ことができます。

### 3. フレンドサーヴァント対応

パーティの6枠には通常の自前サーヴァントだけでなく、**フレンドが貸してくれるサポートサーヴァント**を入れることもあります。  
フレンドサーヴァントについてはスキルレベルや宝具レベルを個別に上書き指定できるように対応する予定です。
